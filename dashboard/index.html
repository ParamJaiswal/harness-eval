📚 RAG Webhook Integration Guide

To evaluate your RAG pipeline's retrieval and faithfulness, set Provider Type to RAG Pipeline. Your endpoint must accept and return the following JSON structure:

POST Request Payload:
{
  "query": "What are the specs of the H100 GPU?",
  "top_k": 5
}
Expected JSON Response:
{
  "answer": "The H100 has 80GB SXM5 HBM3...",
  "contexts": [
    {
      "text": "NVIDIA H100 GPU features 80GB HBM3 memory...",
      "source": "nvidia_specs.pdf",
      "relevance_score": 0.95
    }
  ]
}

Once configured, run the evaluation against standard benchmarks (e.g. rag_faithfulness or rag_retrieval) to test your system.
